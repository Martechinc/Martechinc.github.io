Advanced Molding & Casting Techniques

From 3D Model to Functional Castings: A Technical Guide

Single-Part Mold Process

Step 1: Model Preparation

  1. Import 3D model (STL/OBJ) into CAD software
  2. Create negative space using Boolean operations
    • Offset mold walls by 8-10mm for material thickness
    • Add registration marks for alignment
  3. Design sprue and vent channels for resin flow

Step 2: Mold Fabrication

  1. Print mold using high-temperature filament (ABS/ASA)
  2. Apply 3-5 coats of resin release agent (Stoner E226)
  3. Post-process with acetone vapor smoothing

Two-Part Mold Methodology

Dual Cavity Construction

  1. Split original model along optimal parting line
    • Analyze undercuts using draft analysis tools
    • Offset split plane by 0.5mm for gasket effect
  2. Create alignment pins and sockets
    • 3mm diameter pins spaced 25mm apart
    • 0.2mm clearance for thermal expansion

Casting Procedure

  1. Pre-heat molds to 40°C (104°F)
  2. Mix resin components (1A:1B by volume)
    • Stir for 2 minutes minimum
    • Degas in vacuum chamber at 25Hg
  3. Pour resin through sprue at 45° angle
  4. Clamp mold halves with 15psi pressure

Critical Failure Analysis

Post-casting challenges encountered:

  1. Adhesion Failure: PLA mold Tg point (60°C) vs resin cure temp (85°C)
  2. Differential Shrinkage: 0.6% resin shrinkage caused mechanical locking
  3. Surface Texture: Printed layer lines created micro-anchors

Successful Resolution: Transitioned to PETG molds with PTFE spray coating achieved 95% release success rate

Technical Specifications

  • Mold Material: Prusament PETG (0.15mm layer height)
  • Casting Resin: Smooth-On Sorta-Clear 37
  • Cure Parameters: 4hr @ 25°C or 45min @ 65°C
  • Surface Prep: 320 grit sanding + SEMSOOTH coating
